# axis-init

initialize axis context governance in your project.

## usage

```bash
npx axis-init@latest
```

this will create a `.axis/` directory in your current path with the necessary structure for context governance.

### structure

- `.axis/axis.json`: project configuration.
- `.axis/instructions/`: agent-specific instruction sets.
- `.axis/research/`: shared project research and documents.
